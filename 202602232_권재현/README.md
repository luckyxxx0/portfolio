# Lucky Portfolio

HTML, CSS, basic JavaScript만 사용해 만든 2페이지 포트폴리오 웹사이트입니다. GitHub Pages에서 바로 보이도록 루트에 `index.html`을 두었습니다.

## Pages

- `index.html`: 자기소개 페이지
- `activities.html`: 활동 목록 페이지

## Features

- Modern responsive Material UI 스타일
- 모바일/데스크톱 반응형 레이아웃
- 다크 모드 전환
- 활동 검색 및 카테고리 필터
- 연락처 복사 알림

## Project Structure

```text
.
├── index.html
├── activities.html
├── style.css
├── script.js
├── assets/
│   └── profile-card.png
└── README.md
```

## GitHub Pages

1. 이 저장소를 GitHub에 업로드합니다.
2. GitHub 저장소에서 **Settings > Pages**로 이동합니다.
3. **Build and deployment**에서 source를 `Deploy from a branch`로 선택합니다.
4. Branch를 `main` 또는 `master`, folder를 `/root`로 선택합니다.

저장소 주소: https://github.com/luckyxxx0/portfolio
